# Market Efficiency in College Football Betting

> **JEM207 — Data Processing in Python** | Charles University, Prague

## Overview

This project investigates whether the college football (CFB) betting market is **informationally efficient** — that is, whether betting spreads accurately predict game outcomes or whether systematic biases exist.

Betting markets function similarly to financial prediction markets. In an efficient market, the point spread should be an unbiased predictor of the actual game margin, meaning home teams should cover the spread roughly **50% of the time**. We test this hypothesis across 4 seasons of CFB data (2020–2023), analyzing patterns by conference, week, and spread size.

## Setup & Installation

### 1. Clone the repository

```bash
git clone https://github.com/yourusername/cfb-market-efficiency.git
cd cfb-market-efficiency
```

### 2. Install dependencies

```bash
pip install -r requirements.txt
```

### 3. Get a free API key

This project uses the [College Football Data API](https://collegefootballdata.com).

- Go to [collegefootballdata.com](https://collegefootballdata.com) and register for a free account
- Copy your API key from your account page
- Create a file called `config.py` in the root folder:

```python
API_KEY = "your_api_key_here"
```

> **Note:** `config.py` is listed in `.gitignore` and will never be pushed to GitHub.

### 4. Run the notebook

Open `final_cfb_market_efficiency.ipynb` in Jupyter and run all cells from top to bottom:

```bash
jupyter notebook final_cfb_market_efficiency.ipynb
```

The notebook will automatically:
- Load raw data from `raw_data/` if already downloaded
- Download fresh data from the API if not
- Clean and process the data
- Save `cfb_cleaned_data.csv`
- Run all analysis and generate visualizations

### 5. Run the interactive dashboard

```bash
streamlit run app.py
```

Make sure `cfb_cleaned_data.csv` is in the same folder before running.

## Analysis

### 1. Cover Rate Analysis
How often do home teams cover the spread? An efficient market predicts ~50%. We found home teams cover ~62% of the time — evidence of a systematic home team bias.

### 2. Spread Error Analysis
We analyze the distribution of spread errors (actual margin minus predicted spread) to identify whether the market systematically over- or under-estimates home team performance.

### 3. Spread vs Actual Margin
A scatter plot comparing the betting spread with the actual game margin. A perfectly efficient market would have all points on the diagonal.

### 4. Spread Group Analysis
Games are categorized by spread size (big favorite, small favorite, even, small underdog, big underdog) and cover rates are compared across groups.

### 5. Weekly Analysis
Cover rates broken down by week to check whether market efficiency changes throughout the season.

### 6. Conference Analysis
Cover rates and spread errors are compared across conferences — including a dedicated SEC vs Big Ten comparison to test whether higher-profile, more-watched games are priced more accurately.

### 7. Over/Under Analysis
We test whether the total points market is more efficient than the spread market by analyzing over hit rates by season and conference.

### 8. Biggest Market Surprises
Tables of the games where the market was most wrong — biggest home underdog wins and biggest home favorite flops.

---

## Key Findings

- Home teams cover the spread **~62% of the time** — well above the 50% expected in an efficient market
- The average spread error is **~+12 points**, meaning the market systematically underestimates home team performance
- **Big home favorites (spread ≤ -14) cover ~99% of the time** — large talent gaps dominate home field effects
- The **SEC and Big Ten** show different efficiency patterns compared to smaller conferences
- The **over/under market is more efficient** than the spread market, with hit rates closer to 50%
